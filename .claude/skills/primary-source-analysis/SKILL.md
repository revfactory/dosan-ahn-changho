# 1차 사료 분석

1차 사료는 가장 강한 근거인 동시에 가장 위험한 함정이다. 사료는 사실의 투명한 창이 아니라 특정 시점에 특정 의도를 가진 사람이 만든 인공물이기 때문이다. 일제 신문조서는 강압 아래 작성되었고, 회고록은 수십 년 뒤의 기억과 미화를 담는다. 따라서 1차 사료를 다룰 때는 "무엇이 적혀 있는가"보다 먼저 "누가, 언제, 왜 적었는가"를 물어야 한다. 이 스킬은 그 질문의 절차를 규정한다.

## 1. 사료 비판 (Source Criticism)

모든 사료에 대해 외적 비판 → 내적 비판 순으로 수행하고, 결과를 카탈로그의 `criticism` 필드에 기록하라.

### 외적 비판 — 이 사료는 무엇인가

- **진위:** 원본인가, 사본인가, 후대 편집본인가. 전집·선집에 수록된 텍스트는 편집 과정에서 변형되었을 수 있으므로 가능하면 원본 소재를 추적하라.
- **작성 시점:** 사건 당시 기록인가, 사후 회고인가. 사건과 기록 사이의 시간 거리가 멀수록 기억 왜곡 가능성이 커진다. 작성 시점이 불명이면 그 사실 자체를 기록하라.
- **작성·전승 경로:** 사료가 현재 소장처에 도달한 경로를 알 수 있으면 기록하라. 경로가 끊긴 사료는 진위 판단에 한계가 있음을 명시한다.

### 내적 비판 — 작성자는 왜 이렇게 적었는가

- **일제 측 기록(신문조서·판결문):** 강압적 신문 환경에서 작성되었고, 작성 주체는 피의자를 유죄로 구성할 동기를 가진다. 진술 내용을 사실로 그대로 채택하지 말고 "일제 신문조서에 따르면"이라는 한정과 함께, 작성 주체의 의도를 주석으로 부기하라. 단, 일제 기록은 날짜·장소 같은 행정 사실에는 오히려 정밀한 경우가 많다 — 진술 내용과 행정 기록을 구분해 평가하라.
- **회고록·전기류(이광수 『도산 안창호』 등):** 후대의 존경과 미화가 개입한다. 저자와 도산의 관계(이광수는 수양동우회 동지이자 민족개조론 논쟁의 당사자)를 주석으로 남기고, 회고에만 있는 사실은 1차 사료가 아닌 2차 사료 등급으로 처리하라.
- **자기 기록(일기·서한):** 가장 솔직하지만, 자기 정당화와 검열 의식(서한이 일제에 검열될 가능성을 의식한 표현)이 있을 수 있다. 수신자가 누구인지에 따라 같은 사건의 서술 톤이 달라짐을 유의하라.
- **단체 자체 기록(흥사단 공식 역사 등):** 단체의 정통성 서사가 개입한다. 학술 연구와 서술이 다른 지점을 발견하면 차이를 기록하고 어느 쪽도 임의 채택하지 마라.

내적 비판은 사료를 폐기하기 위한 것이 아니다. 편향을 명시한 사료는 편향을 숨긴 사료보다 훨씬 유용하다 — 후속 검증자가 할인율을 적용할 수 있기 때문이다.

### 사료 유형별 비판 체크포인트

| 유형 | 먼저 물을 것 | 흔한 함정 |
|------|-------------|----------|
| 일기 | 집필 직후 기록인가, 추기(追記)인가 | 후일 정리본을 당일 기록으로 오인 |
| 서한 | 수신자가 누구인가, 검열 의식이 있는가 | 검열 회피 표현을 액면 그대로 해석 |
| 연설문 | 연설 당시 속기인가, 후대 복원인가 | 복원문을 발화 원문으로 인용 |
| 동시대 신문 | 발행 주체의 정치적 위치는 어디인가 | 기관지(공립신보·신한민보)의 자기 단체 서술을 중립 보도로 취급 |
| 신문조서·판결문 | 진술 내용과 행정 사실을 구분했는가 | 강압 진술을 자백 사실로 채택 |
| 회고록·전기 | 저자와 대상의 관계, 집필 시점과의 거리 | 수십 년 뒤의 직접 인용문을 글자 그대로 신뢰 |
| 단체 기록 | 단체의 정통성 서사가 개입하는 지점은 어디인가 | 창립 일자·주도자 서술의 자기중심화 |

## 2. 원문 인용 규칙

- **글자 단위 정확성:** 원문은 한 글자도 바꾸지 말고 그대로 전사하라. 옛 한글 표기·한자·띄어쓰기를 임의로 현대화하지 마라. 전사의 변형은 검증 불가능한 2차 텍스트를 만든다.
- **현대어 병기:** 원문 다음에 현대어 풀이를 병기하라. 형식: 원문 인용 블록 → `(현대어: ...)`. 풀이는 의역이 아닌 직역에 가깝게 — 해석은 풀이가 아니라 별도 분석 문장으로 하라.
- **판독 불가:** 판독 불가능한 글자는 `〔판독불가〕`, 추정 판독은 `〔추정: 글자〕`로 표기하라. 추정을 확정처럼 적으면 오류가 인용 연쇄를 타고 번진다.
- **중략:** 생략은 `〔중략〕`으로 명시하라. 표시 없는 생략은 인용 조작이다.
- **출전 필수:** 모든 인용에 사료 카탈로그 id를 붙여라. 출전 없는 인용은 검증 불가이므로 사용 불가다.
- **해석의 정박:** philosophy-researcher의 사상 해석은 반드시 원문 인용에 정박시켜라. "도산은 ~라고 생각했다"는 문장 하나마다 근거 인용 하나가 대응해야 한다. 인용 없는 해석은 report.md의 가설 절로 격리하라.

### 외국어 사료의 번역 처리

일제 기록(일본어)·미주 기록(영어)·중국 발행물(중문)은 번역 단계가 추가 왜곡원이 된다.

- 원문(원어) 전사를 `transcription`에, 한국어 번역을 `modern_translation`에 넣고 번역 주체(직접 번역/기존 번역본 사용)를 `notes`에 기록하라. 기존 번역본을 쓰면 그 번역본도 출전으로 남겨라.
- 인명·단체명의 원어 표기(예: 安昌浩에 대한 일제 문서의 표기 변형)는 번역하지 말고 원형을 보존해 `notes`에 — 이 표기들이 network-mapping의 `alt_names` 재료가 된다.
- 법률·행정 용어(치안유지법, 예심 등)는 당대 용어를 보존하고 현대 설명을 덧붙여라. 현대 용어로 치환하면 사료의 시대성이 사라진다.

## 3. 사료 카탈로그 스키마

primary-source-researcher의 산출물 `_workspace/01_research/primary-source_catalog.json`은 다음 스키마의 배열이다.

```json
{
  "id": "src-pri-{일련번호}",               // 사료 고유 id. citation-style(S08)의 출처 ID 체계와 호환
  "title": "사료 제목",                     // 예: "안창호 신문조서 (제1회)"
  "type": "diary|letter|speech|newspaper|interrogation|verdict|org_record|photo|etc",
  "author": "작성자",                       // 기관 작성이면 기관명. 미상이면 "미상"
  "date_created": "YYYY-MM-DD",            // 작성일. 불확실하면 연 단위 + 비고
  "language": "ko|ko-hanja|ja|en|zh",      // 원문 언어
  "holder": "소장처",                       // 예: "국사편찬위원회"
  "access_url": "접근 URL 또는 null",       // null이면 location_status로 사유 설명
  "location_status": "confirmed|cited_only|unlocated",  // 원문 확인 / 2차 인용으로만 존재 / 소재 미확인
  "transcription": "원문 전사 (확보 시)",
  "modern_translation": "현대어 풀이 (전사 확보 시)",
  "criticism": {
    "external": "진위·작성 시점·전승 경로 평가 1~3문장",
    "internal": "작성자 의도·편향 평가 1~3문장"
  },
  "related_event_ids": ["evt-..."],         // 이 사료가 근거하는 사건 레코드 id
  "notes": "기타 — 검열 흔적, 편집본 차이 등"
}
```

`related_event_ids`는 사료와 사건을 잇는 유일한 다리다. 이 연결이 없으면 Phase 2에서 "이 사건의 1차 사료가 있는가"라는 질문에 답할 수 없다.

## 4. 발굴·전사 작업 절차

순서를 지키는 이유: 비판 없이 전사부터 하면 편향된 텍스트가 "원문"의 권위를 입고 유통되기 시작하고, 회수 비용은 단계가 내려갈수록 커진다.

1. **소재 파악:** historical-research(S02)의 아카이브 표와 검색 전략으로 사료의 소장처·접근 경로를 확정하라. 학술 논문의 각주가 가장 좋은 소재 단서다.
2. **카탈로그 등재:** 전사 전에 먼저 카탈로그 레코드를 만들어라(외적 비판 필드 포함). 등재가 먼저여야 "전사 중인 사료"와 "미발굴 사료"가 구분된다.
3. **전사:** 2절 규칙대로 글자 단위 전사 + 현대어 병기. 분량이 큰 사료는 도산 관련 부분을 우선 전사하되, 생략 구간을 `notes`에 명시하라.
4. **내적 비판:** 전사를 마친 뒤 전체 맥락에서 작성자 의도를 평가해 `criticism.internal`을 기록하라. 부분만 읽고 내린 비판은 맥락 오독의 위험이 있다.
5. **사건 연결:** 사료가 뒷받침하는 사건 레코드를 찾아 `related_event_ids`를 채워라. 해당 사건이 아직 없으면 담당 연구원에게 수집을 요청하고 그 사실을 report.md에 남겨라.
6. **인계:** philosophy-researcher 등 이 사료를 해석에 사용할 에이전트를 위해, 인용 가능 구간과 비판 요지를 report.md에 요약하라.

## 5. 소재 미확인 사료 처리

존재가 알려졌으나 원문을 확보하지 못한 사료는 흔하다. 이를 처리하는 원칙은 "없는 것을 있는 것처럼 만들지 않기"다.

1. **카탈로그에 등재하되** `location_status: "unlocated"`로 표기하라. 미확인 사료의 존재 기록 자체가 후속 발굴의 단서다.
2. **2차 인용으로 대체할 때는** 그 사실을 명시하라 — `transcription`에 원문 대신 "원문 미확인. ○○○(연도, 면수)의 재인용에 따름"을 적고, 재인용 출처를 `notes`에 기록하라. 재인용 텍스트를 원문 확인된 것처럼 적으면, 재인용자의 전사 오류까지 1차 사료의 권위를 얻게 된다.
3. **재인용의 신뢰도는 한 단계 아래다.** 이 사료에 근거한 사건 레코드의 `confidence`는 A가 될 수 없음을 관련 연구원에게 인계하라.
4. **탐색 경로를 기록하라.** 어느 아카이브를 어떤 검색어로 탐색했는지 report.md에 남겨, 보완 라운드에서 같은 탐색을 반복하지 않게 하라.

## 6. 재호출 시 (사료 분석 다시/카탈로그 업데이트)

기존 카탈로그가 있으면 `id`를 보존한 채 증분 갱신하라 — `src-pri-` id는 citation-style(S08)의 출처 원장이 승계하는 키이므로, id 변경은 인용 체계 전체의 참조를 끊는다. 새 사료는 새 id로 추가, `unlocated`였던 사료의 원문을 확보하면 같은 레코드의 `location_status`를 갱신하고 전사를 채워라. 갱신 사실을 report.md에 남겨 관련 연구원이 confidence 상향을 재검토하게 하라.

## 산출물 검증

제출 전 다음을 확인하라.

1. catalog.json이 유효한 JSON이고, 모든 레코드에 `id`, `title`, `type`, `holder`, `location_status`가 있는가.
2. `location_status: "confirmed"`인 레코드는 `access_url` 또는 소장처 접근 경로가 있는가 — "확인했다"는 주장에 경로가 없으면 미확인과 같다.
3. `transcription`이 있는 모든 레코드에 `modern_translation`이 병기되어 있는가.
4. 모든 레코드의 `criticism.external`과 `criticism.internal`이 비어 있지 않은가 — 비판 없는 사료는 분석되지 않은 사료다.
5. 2차 인용 대체 레코드에 재인용 출처가 명시되어 있고, 원문 확인으로 위장되지 않았는가.
6. `related_event_ids`의 id들이 실제 events.json에 존재하는가 (끊어진 참조 0건).
7. report.md에 탐색 경로(아카이브·검색어)와 미발굴 사료 목록이 기록되어 있는가 — 보완 라운드의 출발점이 된다.
