# 도산 소설 문체·대사 작법 (dosan-novel-style)

도산 안창호 장편소설의 창작 규범이다. 이 작품은 **검증된 역사 위에 세운 대사 중심 소설**이다 — 사실의 뼈대는 `_workspace/02_verified/`가 공급하고, 살과 숨은 대사가 만든다. historical-writing 스킬(한정 문형·[ref:] 마커·건조한 서술)은 **소설 본문에 적용하지 않는다** — 사실 추적은 fact-fiction-ledger가 본문 밖에서 담당하므로, 본문은 소설로서 완전해야 한다.

## 1. 대사 중심 원칙 — 말이 서사를 끈다

- **장면의 추진력은 대사다.** 사건 설명을 지문으로 늘어놓지 말고, 인물들이 그 사건을 두고 다투고 설득하고 망설이는 **말**로 보여줘라. 목표 비율: 장면 분량의 60% 이상이 대사(따옴표 안 + 직결 지문).
- **지문은 무대 지시다.** 표정·동작·침묵·소품(등잔, 찻잔, 신문지면)으로 대사 사이의 공기를 만든다. 내면 서술은 장면당 2~3호흡 이내 — 인물의 속을 말과 행동이 대신 드러내게 하라.
- **대사는 충돌이다.** 같은 편 인물들도 방법론으로 부딪치게 하라(준비론 대 무장론, 통합 대 결별). 모두가 동의하는 대화는 장면이 아니라 회의록이다.
- **연설은 아껴라.** 도산은 연설가지만, 소설이 연설문집이 되면 안 된다. 연설 장면은 부당 1~2회, 그마저 청중 반응·방해·내적 긴장과 교차 편집하라.

## 2. 시대 언어 — 향취는 살리고 가독성은 지킨다

- **호칭이 시대를 만든다.** 호(號)+형/공/선생 체계를 일관 적용하라: 도산이 부를 때 "남강 형"(이승훈), "단재"(신채호), "우남 형"(이승만·관계 악화 후 "우남 박사"로 거리), "성재 형"(이동휘), "백범"(김구— 후배라 "김구 군"→"백범"으로 변천). 도산을 부를 때: 동지들 "도산 형/도산 선생", 일제 심문자 "안창호". 부부간: "당신"(이혜련→도산 서한 실례 기반), 도산→아내 "혜련".
- **어휘는 시대물, 문법은 현대.** "긴하다·상관없소·~하오/~소 체(공적 대화)·~네/~게(친근한 사이)" 같은 어미와 시대 어휘(전보, 여관, 인력거, 학우)는 살리되, 고어 표기·한자 남발로 독해를 막지 마라. 평안도 방언은 향취만 — 어미 한두 개("기래서", "~디요")를 도산 외 평안도 조연에게 절제 사용, 도산 본인은 표준 정중체(전국 청중을 상대한 연설가의 언어).
- **일본어·영어 혼입.** 일제 관헌 장면의 일본어는 한국어로 쓰되 어조로 위압을 표현하고, 꼭 필요한 단어만 원어 병기(「요보」 같은 멸칭은 맥락 신중). 미주 장면의 영어도 동일 원칙.

## 3. 인물 목소리 시트 — 실존의 말에 정박하라

voice-keeper가 `novel/characters/voices.md`에 주요 인물별 시트를 작성하고, scene-writer는 시트 밖의 말투를 쓰지 않는다. 시트 스키마:

```
### {인물명} ({호, 생몰})
- 말의 속도/길이: (예: 도산 — 짧은 질문으로 시작해 긴 설득으로 맺는다)
- 논법: (예: 도산 — 비유 하나 → 구체적 실행 하나. 추상론으로 끝내지 않는다)
- 어휘 지문: 자주 쓰는 말·금기어 (예: 도산 — "무실", "힘", "거짓말 마시오"; 욕설 없음)
- 감정 표현: (예: 도산 — 분노는 침묵과 낮아진 목소리로; 신채호 — 즉발적, 자리를 박참)
- 근거: 실제 연설문·서한·신문조서·회고의 출전 (예: 「개조」(1919)의 문답 구조, 1932 신문 기록의 단문 응답)
- 관계별 변주: 누구에게 어떤 어미를 쓰는가
```

- **도산의 목소리 원천(검증 DB):** 연설 「개조」·육대사 연설·임정 연설(`philosophy.md`), 신문조서 응답 양식, 이혜련·자녀 앞 서한(가정인으로서의 어조), 흥사단 문답식 대화법(질문으로 깨우치는 소크라테스식 — 입단문답 전승). 어록 사용 시 출전 검증 3분류를 따르라: 확인 어록은 그대로, 출전 미확인 어록은 **변형하거나 상황을 새로 지어 쓰되 ledger에 기록**.
- **대사의 시대 고증보다 인물 고증이 우선이다.** 신채호가 도산에게 동의만 하면 그것은 신채호가 아니다 — 관계망(network.json)의 갈등 엣지가 대사의 긴장을 설계하는 지도다.

## 4. 입체적 인물 — 소설에서도 성인전(聖人傳)은 금지

- 도산의 **약점·회의·실패**를 장면화하라: 국민대표회의 결렬의 무력감, 가족을 버려두다시피 한 미안함(이혜련 앞에서의 말 못함), 이상촌의 좌절, 병든 몸. 위인은 결심하는 사람이 아니라 망설인 끝에 움직이는 사람이다.
- 반대편 인물(이승만·창조파·전향자 김희선)도 **자기 논리를 가진 인간**으로 — 악역 만화화 금지. 일제 심문자조차 제 직무의 논리로 말하게 하라.
- 갈등 장면의 사실 근거는 검증 DB의 conflict 엣지·국민대표회의 기록 범위 안에서 — 기록에 없는 악행을 실명 인물에게 창작하는 것은 금지(ledger 스킬 §3 참조).

## 5. 장면·장 구조

- **장면 단위 설계.** 각 장은 2~4개 장면, 장면마다: 시공간(검증 연표 정박) / 등장 인물 / 갈등(누가 무엇을 원하고 누가 막는가) / 전환점 / 다음 장면을 무는 고리.
- **장 말미 훅.** 다음 장을 읽게 만드는 미해결 — 단, 인위적 절단보다 "결정의 무게"로.
- **시간 도약은 장 경계에서만.** 장 내부의 회상은 1회 이내, 대사로 처리 우선("십 년 전 쾌재정에서 내가 뭐라 했는지 기억하시오?").
- **사물 모티프 재사용.** 한 작품을 꿰는 물건들 — 집조(여권), 거국가 악보, 오렌지(전승 채택 시), 송태산장의 돌 — 을 부마다 변주해 장편의 통일감을 만든다.

## 6. 시점·문장

- 기본 시점: **3인칭 제한 — 도산 밀착**. 단 부마다 1~2개 장은 타 인물 시점(이혜련·신채호·심문자 등)을 허용해 도산을 바깥에서 보게 하라 — novel-director가 장별로 확정한다.
- 문장: 짧은 문장이 기본. 묘사는 감각 하나를 정확히(냄새·소리 우선). 부사 절제, "~것이었다" 금지.
- 분량 감각: 장당 4,000~7,000자(한글), 전체 20~24장 + 서장·종장.

## 산출물 검증 (literary-editor·voice-keeper 공용 체크)

1. 장면별 대사 비율 ≥60% (스크립트로 따옴표 행 비율 측정 가능)
2. 시트 밖 말투 0건 — 인물별 어휘 지문·어미 대조
3. 호칭 체계 위반 0건 (관계·시기별 호칭표 대조 — 1932 이후 "우남 형" 같은 시대착오 금지)
4. 연설 장면 부당 ≤2회, 내면 서술 과다 장면 0
5. 도산 약점·갈등 장면이 부마다 ≥1회
6. fact-fiction-ledger의 장면 대장과 본문 장면 1:1 — 대장 없는 장면 0건
