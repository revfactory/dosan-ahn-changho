# 페이지 명세 07 — 인물 (people.html)

> v1.0 (2026-06-06) / 전제: 00_common.md. 구현(그래프 시각화)은 Phase 4 소관 — 본 명세는 콘텐츠·데이터 요구만 정의한다.

## 목적
도산의 관계망을 **갈등 포함 전체상**으로 보여준다. 동지 관계와 갈등 관계를 동일 밀도로 — 우상화 금지의 관계망 판본. "관계가 통설로 떠도나 근거 미달"인 목록(미확정)도 독자 가치로 노출한다.

## 데이터 소스
- 관계망: `network.json` v1.2.1 — 노드 81(인물 59·조직 22), 엣지 135(membership 63·comrade 48·family 10·conflict 8·mentor 3·patron 3)
- 소비 필드: nodes{id,name,hanja,alt_names,birth,death,role,summary}, edges{source,target,type,period,evidence_event_ids,description,period_note}
- **edges_unconfirmed 19건은 그래프에 넣지 않는다** — §5 미확정 표로만
- 본문 서사: `synthesis/people.md` §1–7 (유일 사실 원천)

- 엣지 상세에 description + **evidence_event_ids → timeline 앵커 링크** (근거 없는 관계선은 존재하지 않는다는 것을 보여주는 장치)

## 집필 규칙 (people 특화)
- **#conflicts 7건 전건 수록 의무**: 이승만·이동휘·신채호·김규식·이광수·윤해·(이종호+유동열). 갈등 상대 폄하 문형 금지 — 양측 입장 사실 병기.
- 이혜련은 "도산의 아내" 종속 서술 금지 — 독립운동가 개인(2008 건국훈장 애족장)으로 서술 (people.md §1 문형 유지).
- #unconfirmed 표는 people.md §6을 그대로 옮긴다 (서재필·안중근·박용만·손정도·조만식·김동원·박현환 등) — "왜 못 싣는가" 사유 열 포함. 머리말에 "관계가 통설로 떠도나 근거가 미달인 정보 자체가 독자 가치" 취지 1–2문장.
- 조소앙 출생 연도는 **연 단위(1887) 유지** — network 대기 목록 P-1(1887-05-02 상향)은 동결 해제 후 반영 (데이터·서술 평면 불일치 금지).
