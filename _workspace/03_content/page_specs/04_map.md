# 페이지 명세 04 — 활동 지도 (map.html)

> v1.0 (2026-06-06) / 전제: 00_common.md. 구현은 Phase 4 map-developer(interactive-viz 스킬, Leaflet CDN) 소관 — 본 명세는 콘텐츠·데이터 요구만 정의한다.

## 목적
한반도–미주–중국·연해주–유럽 경유에 걸친 활동 무대의 공간 시각화. 학생 여정(J3)의 "활동 무대 정리"와 이동 서사(망명 대장정)의 직관 전달.

## 데이터 소스
- `timeline.json` 중 **place.lat/lng 보유 + confidence A·B·C인 사건** (D 제외 후 좌표 보유 117건 내외 — 정확 수는 데이터 변환 시 확정, null 좌표는 마커 생략이 계약)
- 이동 경로: tags에 '이동' 포함 사건 38건의 시간순 좌표열

## 화면·콘텐츠 요구
1. **도입문** (narrative-writer, 300–500자): 지도가 보여주는 것(검증 사건 중 위치 확인분만), 좌표 미상 사건은 표시되지 않음을 고지, 지명은 당대 지명+현대 지명(place.modern_name) 병기 원칙.
2. **마커**: 사건 단위. 팝업 = 제목+날짜(precision 표기)+summary 1–2문장+`timeline.html#evt-id` 링크. 등급 배지·disputed 마크는 연표와 동일 규칙.
3. **경로 해설** (narrative-writer, 300–500자 — 도입문과 합산 600–1,000자): 주요 이동 경로 5건의 폴리라인 해설 캡션.
   | 경로 | 연대 | 근거 사건 (timeline.json) |
   |------|------|--------------------------|
   | R1 도미 | 1902 | evt-early-023 → evt-amer-002 (경유지 항로는 C — "전해진다" 캡션) |
   | R2 귀국 | 1907 | evt-shin-002~004 |
   | R3 망명 대장정 | 1910–1911 | evt-shin-019~028 (행주→웨이하이웨이→칭다오→블라디보스토크→시베리아 횡단→뉴욕) |
   | R4 재방문 왕복 | 1924–1926 | evt-amer-027 → evt-provgov-021 |
   | R5 압송 | 1932 | evt-provgov-029~030 → evt-hsd-011 (상하이→서울) |
   - 경로 좌표는 근거 사건의 좌표만 잇는다 — 사건이 없는 중간 기착지를 임의 보간하지 않는다. 캡션의 사실 문장에 [ref:] 부착.
4. **시기 필터**: P1–P8 (sitemap §3) — 연표와 동일 체계.

## 분량
도입문+경로 해설 합산 600–1,000자.

## 이미지 슬롯
없음 (필수 0). 지도 타일이 시각 요소다.

## 교차링크
- 마커 팝업 → `timeline.html#evt-...` (전 마커 의무)
- 경로 해설 → `life.html#ch-07`(R3), `#ch-13`(R5) 등 해당 장

## 완성 기준
- 마커 수 = D 제외 후 좌표 보유 사건 수와 일치 (임의 추가·누락 0)
- null 좌표 사건의 유령 마커 0
- 경로 5건 전부 근거 사건 좌표만 사용 (보간점 0)
- 팝업 → 연표 앵커 링크 100%
