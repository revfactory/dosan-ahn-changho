# 페이지 명세 09 — 갤러리 (gallery.html)

> v1.0 (2026-06-06) / 전제: 00_common.md. **visual-curator의 1차 작업 명세** — 슬롯 집계표(§3)가 사이트 전체 수집 범위다.

## 목적
사이트 전체 이미지의 집합 뷰. 모든 이미지는 어딘가의 페이지 슬롯에서 왔거나 갤러리 고유분이며, **매니페스트(메타데이터) 없는 이미지는 존재하지 않는다.**

## 데이터 소스
- visual-curator가 작성하는 이미지 매니페스트(visual-sourcing 스킬 스키마 — Phase 4에서 `site/data/images.json`으로 변환)
- 매니페스트 필수 필드(콘텐츠 평면 요구): 이미지 id, 슬롯 id 목록(img-{page}-{NN}), 시기 코드(P1–P8), 캡션 4요소(00_common §7), 사용 페이지 목록, 라이선스·출처

## 섹션 구조
1. **도입** (narrative-writer, 200–400자): 수록 기준(퍼블릭 도메인/자유 라이선스 검증 통과분만), 캡션의 검증 원칙, 연출·재구성 이미지 부재 고지.
2. **시기별 그리드** — P1–P8 탭/필터 (sitemap §3 체계). 각 이미지 카드: 이미지 + 캡션 4요소 + **사용 페이지 역링크** (워크스루 수정 #1) + 시기 코드.
3. **문서·지면 섹션** — 사진과 문서(신문 지면·공문서·문서 사료)를 유형 필터로 구분.
