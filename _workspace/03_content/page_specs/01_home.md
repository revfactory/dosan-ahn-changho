1. **히어로** — 사이트명 + 한 줄 정의("도산 안창호(1878–1938) 일대기 — 검증된 기록으로 읽는") + 대표 초상 (img-home-01). **어록·명언 카피 배치 금지** — 인용 가능 어록은 전 사이트 2건뿐이고(philosophy #sec-8 ①) 그마저 한정 표기 의무라 히어로 카피로 부적합, 유명 어록 대다수는 D등급 노출 금지다

3. **이 사이트가 다루는 방식** (300–450자) — 등급 체계(A/B/C/D)·상충 병기·공백 공개 원칙 요약 + `references.html#methodology` 링크. "확인되지 않은 유명 일화·어록은 싣지 않았다"를 명시하고 `archives.html#excluded-traditions` 링크
4. **탐색 카드 9장** — 생애/연표/지도/조직/사상/인물/사료/갤러리/참고문헌. 카드당 한 줄 소개(데이터 규모 명시 — 예: "사건 165건의 인터랙티브 연표")

6. **검증 통계 박스** — 사건·주장·인물·사료 수치 + 등급 분포 → references 링크
